## Decisions Made

- **Bumped version to 1.0.0** (was 0.1.0). The classical methods + evaluation
  suite constitute a complete, usable release per PI direction. Classifier
  changed from Alpha to Beta (PyPI convention — Beta means usable but the API
  may still evolve).
- **Removed `deep` and `llm` optional-dependency groups** from pyproject.toml
  entirely rather than leaving them as empty placeholders. They'll be re-added
  when the methods are actually extracted in v1.1. Shipping uninstallable
  dependency stubs for non-existent code is misleading.
- **Did NOT strip notebook outputs.** The PHI scan found real patient data
  (patient IDs, dates of birth, medical record numbers) in notebook output
  cells — particularly `Data_linking_REDCap_TSQIC_ACCESS_oper_based_updated.ipynb`
  and the Timeseries imputation notebooks. However, the git-history scrub
  (required before making the repo public) is a PI-owned gate. The PR to main
  can land while the repo is private; notebooks must be addressed before the
  repo goes public.
